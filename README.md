# Car Management System

## Overview

Car Management System is a C# console application designed to manage a vehicle inventory.
The system allows users to add cars, track inventory, sell vehicles, calculate total inventory value, and display the current inventory.

---

## Features

* Add a car to inventory
* Maintain and update vehicle inventory
* Sell a car and remove it from inventory
* Calculate total inventory value
* Display all available vehicles

---

## Architecture & Design

The application follows Object-Oriented Programming (OOP) principles:

* Interface (IAutomobile)
  Defines the contract for all automobile types.

* Abstract Class
  Implements shared functionality and enforces structure for derived classes.

* Concret Class (Car, Truck)
  Inherit from the abstract class and provide specific implementations.

* **Program Entry (Program.cs)
  Handles user interaction through a console-based menu system.
